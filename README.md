# Plain-O Brewing Co.

> *Beer doesn't need a story. Just a color.*

Static website for **Plain-O Brewing Co.**, a craft brewery that keeps things simple — beers named by color, no hype required.

## Pages

| Page | Path |
|------|------|
| Home | `index.html` |
| Taplist | `pages/taplist.html` |
| Events | `pages/events.html` |
| About | `pages/about.html` |
| Contact | `pages/contact.html` |

## Project Structure

```
plaino-brewing/
├── index.html              # Home page
├── pages/
│   ├── taplist.html        # Current beer list
│   ├── events.html         # Events calendar
│   ├── about.html          # Brand story
│   └── contact.html        # Location, hours, contact info
├── css/
│   ├── style.css           # Main stylesheet
│   └── calendar.css        # Events calendar styles
├── js/
│   ├── nav.js              # Mobile nav toggle
│   ├── events-data.js      # Events data
│   └── calendar.js         # Events calendar logic
└── Plainobrewing_logo.png  # Brand logo
```

## Tech Stack

- Plain HTML, CSS, JavaScript — no frameworks, no build step
- [Inter](https://fonts.google.com/specimen/Inter) via Google Fonts
- Google Analytics (G-H10JSFPHW4)
- Hosted via GitHub Pages (custom domain via `CNAME`)
